# Othello

모던하고 경쾌한 스타일의 Othello 보드 게임입니다.

## Features
- 8x8 보드, 기본 규칙 적용
- AI 난이도 3단계 (쉬움/보통/어려움)
- 어려움 미니맥스 깊이 조절(2/3/4)
- 동전 던지기로 흑/백 랜덤 결정
- AI 생각 중 보드 잠금 + 애니메이션
- 착수/뒤집기/턴 전환 사운드 및 미세 이펙트

## How To Run
- `index.html`을 브라우저로 열면 바로 플레이됩니다.

## Controls
- `새 게임`: 흑/백을 랜덤으로 다시 결정하고 리셋
- `힌트 끄기`: 가능 수 표시 토글
- `AI 난이도`: 쉬움/보통/어려움 선택
- `어려움 깊이`: 미니맥스 깊이 선택(2/3/4)

## Deploy (Vercel)
- 이 프로젝트는 정적 사이트입니다.
- `vercel.json`이 포함되어 있어 바로 배포 가능합니다.

## Notes
- 실시간 대전은 서버/리얼타임 서비스가 필요합니다.
